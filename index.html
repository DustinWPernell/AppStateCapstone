The Class

This project started shortly before my capstone class as a side project. As I was going to be required to work on my capstone during class, I opted to postpone any more work until I was in class.

I chose to do this project with a friend who had a similar intrest in my idea. We planned to make an application that was free and easy to use to store Magic The Gathering Cards and Decks.
